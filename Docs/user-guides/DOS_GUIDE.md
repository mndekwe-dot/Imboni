# Director of Studies (DOS) Guide

Academic oversight — scheduling, results approval, and analytics. New here? See
[Getting started](README.md) first.

## Your sections

| Section | What it's for |
|---|---|
| **Dashboard** | Academic overview and things needing attention. |
| **Students** | The student roll; search and view profiles. |
| **Teachers** | Teaching staff and their subject/class assignments. |
| **Results** | Review and **approve** results submitted by teachers. |
| **Scheduling / Timetable** | Build and adjust the class timetable. |
| **Exams** | Plan the exam schedule. |
| **Attendance** | School-wide attendance overview. |
| **Analytics** | Performance trends and charts. |
| **Student Leaders** | Manage prefects / student leadership. |
| **Announcements / Messages** | Communicate with staff and students. |
| **Settings** | Academic settings (terms, etc.). |

## Common tasks

**Approve results**
Open **Results**. Teachers' submitted marks wait here for review — check them and
**approve** to publish to students and parents (or send back if something's off).

**Build or adjust the timetable**
Open **Timetable**. Add a lesson to a slot, or **drag a lesson** to move it to a
different day/period. If the move clashes with a teacher's or class's existing
lesson, Imboni warns you and lets you confirm or cancel.

**Plan exams**
Open **Exams** to lay out the exam timetable across days without class clashes.

**Look at performance**
Open **Analytics** for trends by class, subject, or term.

**Set up a new term / academic settings**
Open **Settings** to manage academic terms and related configuration.

## Onboarding a lot of data at once
For setting up classes, timetables, or importing many people at the start of a
term, the school's administrator can use the bulk-import tools — see the
[Admin guide](ADMIN_GUIDE.md) and `Backend/onboarding_samples/`.

## Tips
- Clear the **Results approval** queue regularly so families see marks promptly.
- Use the timetable **drag-to-move** for quick fixes; the clash check protects you.
